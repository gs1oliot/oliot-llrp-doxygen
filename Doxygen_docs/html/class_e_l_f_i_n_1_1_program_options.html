int ELFIN::ProgramOptions::processCmdlineOptions ( int  argc,
char *  argv[] 
)

Parse the command line and configuration file options, and store them.

That class is intended to store values of options, and can store values of arbitrary types. Next, the calls to store, parse_command_line and notify functions cause vm to contain all the options found on the command line.
