Detailed Description

Abstraction for physical RFID reader.

Remarks
The StubReader is designed to run on single thread. So every methods that utilize StubReader object should gain lock of _pReaderLock.

Parameters

Warning
If the matching tag exists, this method updates the tag with current information using StubAntenna::updateExistingStubTag()
