Detailed Description

Inherits LLRP::CConnection class. It has additional methods for Reader role.

Remarks
Additional methods are moved to CConnection of LTK. Now, this class just acts similar to the proxy of LLRP::CConnection class.

int ELFIN::CConnectionFnC::closeConnectionToFnC ( int  __isReaderInitiated)

Close connection to the FnC server.

Remarks
If __isReaderInitiated = 0, then close listening socket and communication socket.
If not, just close communication socket.
The method that closes communication socket is closeConnectionToReader(adopted from LTK), but what it does is just closing the socket.
